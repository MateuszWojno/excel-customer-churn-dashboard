# Customer Churn Analysis Dashboard (Excel)

## Business Problem
The goal of this project was to analyze customer churn in a banking dataset and identify key drivers of customer attrition.

## Data
The dataset contains customer demographics, account balances, product usage, and activity metrics.

## Tools & Technologies
Excel, Power Query, Power Pivot, Data Visualization, Business Analytics

## Process
- Data cleaning and transformation in Power Query  
- Handling missing values and correcting data types  
- Building pivot tables and KPI calculations  
- Designing an interactive dashboard with slicers and visuals

## Key Insights
- Higher churn observed among inactive customers  
- Customers with low balances showed significantly higher churn risk  
- Certain product groups exhibited elevated churn levels
